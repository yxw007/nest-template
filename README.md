## ✨ Features

- [x] Support: Multi-environment configuration
- [x] Integration: JWT
- [x] Supported: Log blocker and HTTP exception blocker
- [x] Supported: HTTP exception filter
- [x] Integration: MySQL
- [x] Integration: Request parameter verification
- [x] In the tank：Scheduled tasks
- [ ] Perfect: Test cases

## 🛠️ Install dependencies

```bash
$ npm install
```

## 🚀 Compile and run

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## 🧪 Run the test case

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```

## 🚢 Deploy

```bash
$ yarn install -g mau
$ mau deploy
```
